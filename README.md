# OPD Example
# 
# To setup
```
> git clone https://github.com/openpolicedata/OPD_Explorer.git
> cd OPD_Explorer/
> pip install -r requirements.txt
```

# To run
```
> cd OPD_Explorer/
> streamlit run opd_download_page.py
```

# To upgrade streamlit
```
> pip install streamlit --upgrade
```
